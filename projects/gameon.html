Game On

Mobile devices are nowadays “Swiss Army Knives”. They support a wide range of tasks, enabling people access a wealth of information and services through their extensive connectivity; they have the potential to empower people in everyday tasks. Unfortunately, getting started with a smartphone is not an easy task, particularly for blind users. The tutorials and help systems that come out of the box are not enough for a user to even get started. This masters is aimed to address the problems people face when first interacting with a new mobile device by gamifying mobile usage to motivate users to explore and repeat tasks, learning everything from gestures to tasks.

This project will involve working with Android.

You will explore novel ways to present getting started tutorials inspired in game design approaches (e.g.https://www.youtube.com/watch?v=dBmIkEvEBtA ). You will develop a system that allows the user to play while learning the basic tasks (e.g. transform the contact application in a level that the user has to travel through without changing its interface undeneath). The user is always playing the game as long as he is interacting with the device. You will implement and explore several gamification components: daily quests for the user to perform (e.g. “write a 100 charaters SMS”), achievement system (e.g. “You are now an Internet Guru”), prompt users to new quests to explore more of the device, progression mechanisms among others. You will add social features to the system allowing users to share their accomplishements or even challenge other “players/users”. Together with the team members you will conduct user studies with end users (e.g. blind, novice) to understand how and when to prompt new tasks assessing what is its impact in mobile usage and learning curve.
