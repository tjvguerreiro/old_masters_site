Puppet Master

Mobile devices are part of our daily lives. Unfortunately, learning how to use them still poses challenges for many users. Moreover, every new app can provide additional difficulties the users are not prepared to handle. These difficulties are exacerbated in user groups that face additional challenges (novice, blind, elderly, and motor impaired). To overcome the barriers, users tend to rely on the help of friends, family and colleagues. However, assistance is not always at hand, which at times can even render the device unusable. This project is aimed at exploring real-time remote assistance between users on mobile devices.

This project will involve working with Android and, eventually, crowd sourcing platforms. You will develop an Android system composed of two main components: one application that allows users to request help and control the flow of the remote assistance; a second that allows friends and family to receive a request and remotely provide assistance. This assistance can be anything from remotely control the device to providing clues to the user, we will explore this design space. In the next step you will integrate crow sourcing platforms allowing for crowdworkers to receive and answer requests. Together with the team members you will work closely with end users (e.g. blind, novice) to understand how assistance can be provided to maintaing the user confortable, in control of their device while enabling learning.

The student working on this project will work in close collaboration with André Rodrigues, a PhD student, that is working on ways to improve the usage and learning of mobile devices by novice users and those with different abilities. This project is done in collaboration with researchers from the University of Newcastle, UK, and Técnico, University of Lisbon.
